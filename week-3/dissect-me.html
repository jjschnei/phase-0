Oh crap, I was trying to add some changes that were suggested by my coworkers in Bangladesh, Brazil, and Turkey. But, now I can't find the documents that they sent me. They are gone forever!

You should really use a Version Control System for projects like this. The most popular exmample of it is Git. We use it all the time in software development. It's a life saver! You first have to memorize a few odd phrases in order to use it in your computer's command line, but once you get the hang of it, you'll want to use it everywhere!

I use it all the time cooking! I start the recipe on what I call the [master branch].

Then, when I want to experiment with some new flavors, I can make additional branches to toy with by pressing [git checkout -b new-branch-name]

Why Use GitHub?

GitHub stores your code “in the cloud”. This makes it easy for distributed teams to work on the same project because they can access it from anywhere over the Internet. This makes GitHub an ideal version control tool for both individuals and distributed teams. In addition to hosted version control and source code management, GitHub also provides social networking functionality. Users have profiles that can show their work and contributions. Projects can be discussed publicly to maximize collaboration.
